Class TestHFileOutputFormat

  • java.lang.Object
    • org.apache.hadoop.hbase.mapreduce.TestHFileOutputFormat

  • public class TestHFileOutputFormat
extends Object
    Simple test for KeyValueSortReducer and HFileOutputFormat. Sets up and runs a mapreduce job that writes hfile output. Creates a few inner classes to implement splits and an inputformat that emits keys and values like those of PerformanceEvaluation.
