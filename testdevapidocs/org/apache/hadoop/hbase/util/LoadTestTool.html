    @InterfaceAudience.LimitedPrivate(value="Tools")
public class LoadTestTool
extends org.apache.hadoop.hbase.util.AbstractHBaseTool
    A command-line utility that reads, writes, and verifies data. Unlike PerformanceEvaluation, this tool validates the data written, and supports simultaneously writing and reading the same set of keys.
