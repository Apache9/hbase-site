Class ProcessBasedLocalHBaseCluster

  • java.lang.Object
    • org.apache.hadoop.hbase.util.ProcessBasedLocalHBaseCluster

  • public class ProcessBasedLocalHBaseCluster
extends Object
    A helper class for process-based mini-cluster tests. Unlike MiniHBaseCluster, starts daemons as separate processes, allowing to do real kill testing.
