      • testGetSplits

        public void testGetSplits()
                   throws IOException,
                          InterruptedException,
                          ClassNotFoundException
        Tests a MR scan using specific number of mappers. The test table has 25 regions, and all region sizes are set as 0 as default. The average region size is 1 (the smallest positive). When we set hbase.mapreduce.input.ratio as -1, all regions will be cut into two MapRedcue input splits, the number of MR input splits should be 50; when we set hbase .mapreduce.input.ratio as 100, the sum of all region sizes is less then the average region size, all regions will be combined into 1 MapRedcue input split.
