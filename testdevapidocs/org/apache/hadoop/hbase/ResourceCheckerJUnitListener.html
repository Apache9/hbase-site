    public class ResourceCheckerJUnitListener
extends org.junit.runner.notification.RunListener
    Listen to the test progress and check the usage of: - threads - open file descriptor - max open file descriptor

    When surefire forkMode=once/always/perthread, this code is executed on the forked process.
