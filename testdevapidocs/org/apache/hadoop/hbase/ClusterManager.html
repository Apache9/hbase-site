org.apache.hadoop.hbase

Interface ClusterManager

  • All Superinterfaces:
    org.apache.hadoop.conf.Configurable
    All Known Implementing Classes:
    HBaseClusterManager, RESTApiClusterManager

    @InterfaceAudience.Private
interface ClusterManager
extends org.apache.hadoop.conf.Configurable
    ClusterManager is an api to manage servers in a distributed environment. It provides services for starting / stopping / killing Hadoop/HBase daemons. Concrete implementations provide actual functionality for carrying out deployment-specific tasks.
