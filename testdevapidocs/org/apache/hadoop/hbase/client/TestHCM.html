      • testOperationTimeout

        public void testOperationTimeout()
                          throws Exception
        Test that an operation can fail if we read the global operation timeout, even if the individual timeout is fine. We do that with: - client side: an operation timeout of 30 seconds - server side: we sleep 20 second at each attempt. The first work fails, the second one succeeds. But the client won't wait that much, because 20 + 20 > 30, so the client timeouted when the server answers.
        Throws:
        Exception
