Class ByteBufferedKeyOnlyKeyValue

  • java.lang.Object
    • org.apache.hadoop.hbase.ByteBufferedCell
      • org.apache.hadoop.hbase.ByteBufferedKeyOnlyKeyValue
  • All Implemented Interfaces:
    Cell

    @InterfaceAudience.Private
public class ByteBufferedKeyOnlyKeyValue
extends ByteBufferedCell
    This is a key only Cell implementation which is identical to KeyValue.KeyOnlyKeyValue with respect to key serialization but have its data in the form of Byte buffer (onheap and offheap).
