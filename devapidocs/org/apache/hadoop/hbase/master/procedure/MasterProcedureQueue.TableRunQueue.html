    private static class MasterProcedureQueue.TableRunQueue
extends MasterProcedureQueue.AbstractRunQueue
    Run Queue for a Table. It contains a read-write lock that is used by the MasterProcedureQueue to decide if we should fetch an item from this queue or skip to another one which will be able to run without waiting for locks.
