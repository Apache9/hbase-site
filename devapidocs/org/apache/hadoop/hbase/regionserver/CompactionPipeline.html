Class CompactionPipeline

  • java.lang.Object
    • org.apache.hadoop.hbase.regionserver.CompactionPipeline

  • @InterfaceAudience.Private
public class CompactionPipeline
extends Object
    The compaction pipeline of a CompactingMemStore, is a FIFO queue of segments. It supports pushing a segment at the head of the pipeline and pulling a segment from the tail to flush to disk. It also supports swap operation to allow the compactor swap a subset of the segments with a new (compacted) one. This swap succeeds only if the version number passed with the list of segments to swap is the same as the current version of the pipeline. The pipeline version is updated whenever swapping segments or pulling the segment at the tail.
