Interface Shipper

  • All Known Subinterfaces:
    HFileScanner, KeyValueScanner, RegionScanner
    All Known Implementing Classes:
    CollectionBackedScanner, DefaultMemStore.MemStoreScanner, HFileReaderImpl.EncodedScanner, HFileReaderImpl.HFileScannerImpl, HRegion.RegionScannerImpl, KeyValueHeap, MobCompactionStoreScanner, MobStoreScanner, NonLazyKeyValueScanner, NonReversedNonLazyKeyValueScanner, ReversedKeyValueHeap, ReversedMobStoreScanner, ReversedRegionScannerImpl, ReversedStoreScanner, StoreFileScanner, StoreScanner

    @InterfaceAudience.Private
public interface Shipper
    This interface denotes a scanner as one which can ship cells. Scan operation do many RPC requests to server and fetch N rows/RPC. These are then shipped to client. At the end of every such batch shipped() will get called.
