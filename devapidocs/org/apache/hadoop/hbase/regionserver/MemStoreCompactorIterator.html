    Iterator<Cell>

    @InterfaceAudience.Private
public class MemStoreCompactorIterator
extends Object
implements Iterator<Cell>
    The MemStoreCompactorIterator is designed to perform one iteration over given list of segments For another iteration new instance of MemStoreCompactorIterator needs to be created The iterator is not thread-safe and must have only one instance in each period of time
