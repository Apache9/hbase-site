    @InterfaceAudience.Private
public class MobCompactionStoreScanner
extends StoreScanner
    Scanner scans the MOB Store. Coalesce KeyValue stream into List<KeyValue> for a single row. It's only used in the compaction of mob-enabled columns. It outputs the normal cells and delete markers when outputDeleteMarkers is set as true.
