      • passesBloomFilter

        boolean passesBloomFilter(Scan scan,
                        SortedSet<byte[]> columns)
        Checks whether the given scan passes the Bloom filter (if present). Only checks Bloom filters for single-row or single-row-column scans. Bloom filter checking for multi-gets is implemented as part of the store scanner system (see StoreFileScanner#seekExactly) and uses the lower-level API passesGeneralRowBloomFilter(byte[], int, int) and passesGeneralRowColBloomFilter(Cell).
        Parameters:
        scan - the scan specification. Used to determine the row, and to check whether this is a single-row ("get") scan.
        columns - the set of columns. Only used for row-column Bloom filters.
        Returns:
        true if the scan with the given column set passes the Bloom filter, or if the Bloom filter is not applicable for the scan. False if the Bloom filter is applicable and the scan fails it.
