Class ScannerCallable

  • java.lang.Object
    • org.apache.hadoop.hbase.client.RegionServerCallable<Result[]>

    ClientSmallScanner.SmallScannerCallable, ReversedScannerCallable

    @InterfaceAudience.Private
public class ScannerCallable
extends RegionServerCallable<Result[]>
    Scanner operations such as create, next, etc. Used by ResultScanners made by Table. Passed to a retrying caller such as RpcRetryingCaller so fails are retried.

      • isHeartbeatMessage

        protected boolean isHeartbeatMessage()
        Returns:
        true when the most recent RPC response indicated that the response was a heartbeat message. Heartbeat messages are sent back from the server when the processing of the scan request exceeds a certain time threshold. Heartbeats allow the server to avoid timeouts during long running scan operations.
