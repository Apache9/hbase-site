org.apache.hadoop.hbase.wal

Class WALKey

  • java.lang.Object
    • org.apache.hadoop.hbase.wal.WALKey
  • All Implemented Interfaces:
    Comparable<WALKey>, SequenceId
    Direct Known Subclasses:
    HLogKey

    @InterfaceAudience.LimitedPrivate(value="Replication")
public class WALKey
extends Object
implements SequenceId, Comparable<WALKey>
    A Key for an entry in the WAL. The log intermingles edits to many tables and rows, so each log entry identifies the appropriate table and row. Within a table and row, they're also sorted.

    Some Transactional edits (START, COMMIT, ABORT) will not have an associated row. Note that protected members marked @InterfaceAudience.Private are only protected to support the legacy HLogKey class, which is in a different package.
