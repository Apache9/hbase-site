Class NamespaceStateManager

  • java.lang.Object
    • org.apache.hadoop.hbase.namespace.NamespaceStateManager

  • @InterfaceAudience.Private
 class NamespaceStateManager
extends Object
    NamespaceStateManager manages state (in terms of quota) of all the namespaces. It contains a cache which is updated based on the hooks in the NamespaceAuditor class.
