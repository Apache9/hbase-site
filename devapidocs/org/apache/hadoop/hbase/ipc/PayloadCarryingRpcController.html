Class PayloadCarryingRpcController

  • java.lang.Object
    • org.apache.hadoop.hbase.ipc.PayloadCarryingRpcController
  • All Implemented Interfaces:
    com.google.protobuf.RpcController, CellScannable
    Direct Known Subclasses:
    DelegatingPayloadCarryingRpcController

    @InterfaceAudience.Private
public class PayloadCarryingRpcController
extends Object
implements com.google.protobuf.RpcController, CellScannable
    Optionally carries Cells across the proxy/service interface down into ipc. On its way out it optionally carries a set of result Cell data. We stick the Cells here when we want to avoid having to protobuf them (for performance reasons). This class is used ferrying data across the proxy/protobuf service chasm. Also does call timeout. Used by client and server ipc'ing.
