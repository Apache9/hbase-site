org.apache.hadoop.hbase.io

Class ByteBufferPool

  • java.lang.Object
    • org.apache.hadoop.hbase.io.ByteBufferPool

  • @InterfaceAudience.Private
public class ByteBufferPool
extends Object
    Like Hadoops' ByteBufferPool only you do not specify desired size when getting a ByteBuffer. This pool keeps an upper bound on the count of ByteBuffers in the pool and a fixed size of ByteBuffer that it will create. When requested, if a free ByteBuffer is already present, it will return that. And when no free ByteBuffer available and we are below the max count, it will create a new one and return that.

    Note: This pool returns off heap ByteBuffers by default. If on heap ByteBuffers to be pooled, pass 'directByteBuffer' as false while construction of the pool.

    This class is thread safe.
