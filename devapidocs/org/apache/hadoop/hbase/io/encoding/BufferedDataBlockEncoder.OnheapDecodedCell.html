Class BufferedDataBlockEncoder.OnheapDecodedCell

  • java.lang.Object
    • org.apache.hadoop.hbase.io.encoding.BufferedDataBlockEncoder.OnheapDecodedCell
  • All Implemented Interfaces:
    Cell, HeapSize, SettableSequenceId, Streamable
    Enclosing class:
    BufferedDataBlockEncoder

    protected static class BufferedDataBlockEncoder.OnheapDecodedCell
extends Object
implements Cell, HeapSize, SettableSequenceId, Streamable
    Copies only the key part of the keybuffer by doing a deep copy and passes the seeker state members for taking a clone. Note that the value byte[] part is still pointing to the currentBuffer and represented by the valueOffset and valueLength
