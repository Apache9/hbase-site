                          float compactionRatio,
                          long compactionMergeableSize)
        Parameters:
        fileStatus - The current FileStatus.
        compactionRatio - compactionRatio the invalid ratio. If there're too many cells deleted in a mob file, it's regarded as invalid, and needs to be written to a new one. If existingCellSize/fileSize < compactionRatio, it's regarded as a invalid one.
        compactionMergeableSize - compactionMergeableSize If the size of a mob file is less than this value, it's regarded as a small file and needs to be merged
