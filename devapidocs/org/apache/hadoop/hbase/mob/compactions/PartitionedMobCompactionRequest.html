Class PartitionedMobCompactionRequest

  • java.lang.Object
    • org.apache.hadoop.hbase.mob.compactions.MobCompactionRequest

  • @InterfaceAudience.Private
public class PartitionedMobCompactionRequest
extends MobCompactionRequest
    An implementation of MobCompactionRequest that is used in PartitionedMobCompactor. The mob files that have the same start key and date in their names belong to the same partition.
